MY PROJECTS

1) FACE RECOGNITION SYSTEM

In this project I have used Local Binary Patterns Histograms (LBPH) algorithm to train the classifiers.

How This Code Works?

•)Person Details:- Purpose: This function is designed to collect and store individuals' information in a database. Additionally, it captures the person's image for training purposes.
•)Train Data:- Purpose: Clicking this button opens a new window where data is trained using the Local Binary Patterns Histograms method. The trained data is then stored in the classifiers.xml file.
•)Photos:- Purpose: By clicking this button, all the photos taken are displayed.
•)Scan Face:- Purpose: Upon clicking this button, a new window opens for real-time scanning of a person's face. The details of the scanned face are displayed in real-time.

2) AI ASSISTANT FOR PC

•) Project Overview:
In this project, I utilized the Python language to create an AI ASSISTANT FOR PC, aiming to assist with various day-to-day tasks.
•)Functionality:
The AI assistant can perform a range of tasks such as opening applications, telling jokes, and reading news.
•)API Integration:
API keys have been incorporated to enable news reading, and additional API keys can be added to enhance the overall user experience and functionality.
•)Application Control:
The assistant is capable of opening applications like Word, PowerPoint, Google, etc.
•)Web Browsing:
Users can instruct the assistant to open commonly used websites or specify a particular website of their choice. Wikipedia can also be accessed through the program by stating "search Wikipedia for + query."
•)Mathematical Operations:
Basic mathematical operations such as addition, subtraction, multiplication, and division can be performed using the assistant.






3)LOGIN PAGE WITH OTP (2 STEP VERIFICATION)

•) This website allows you to perform 2-step verification. When logging in, you will need to enter an OTP that will be sent to your email.
•) During the sign-up process on the website, for enhanced security, the password is hashed using the "SHA-256" algorithm and stored in the database.
•) When logging in on the website, it will first check the entered password against the hashed password in the database. If the passwords match, you will receive an OTP in the email you used for signing up.
